# 1.Signup Page Documentation

# Introduction
This README file provides a detailed explanation of the Signup Page structure and styling. The page is created using HTML and CSS, allowing new users to create an account by entering information such as their name, email, mobile number, password, and user type.

# 2 Page Structure Overview
2.1 Head Section
The head section contains:
Metadata for proper display on all device sizes
The title of the page
Internal CSS used to style all elements of the signup form
2.2 Body Section
The body includes:
A centered outer box that holds the entire signup form
Input fields for user details
A dropdown for selecting the user type
A button for submitting the signup form
A message and login link for existing users

# 3. Styling Description (CSS)
3.1 Body Styling
The default font for the page is Arial
The background is white to maintain a clean and simple appearance
3.2 Outer Box Styling
The form container has a fixed width to keep the layout compact
It is centered using auto margins and spaced downward from the top
Padding inside the box provides spacing between elements
A white background and rounded corners enhance the design
A shadow makes the box visually stand out
3.3 Heading Styling
The heading "Signup" is centered
A purple color is used to highlight the title
Extra spacing is added below the heading
3.4 Input Field Styling
Input fields span the full width of the form
Padding provides comfort when typin
Margins separate each field vertically
Light grey borders and rounded corners give a modern look
Placeholders guide the user on what to enter
3.5 Dropdown (User Type) Styling
The dropdown is centered inside the form
It has a smaller width than input fields for visual distinction
Padding and rounded corners improve appearance
Extra spacing aligns it properly with other fields
3.6 Button Styling
The button spans the full width for easy clicking
Uses a pale pink background with black text
Rounded corners give it a smooth appearance
On hover, the background changes to a lighter pink
A pointer cursor indicates the button is clickable
3.7 Text and Link Styling
The message "Already have an account?" is centered
The login link is centered and styled in blue
Displayed as a block element for proper spacing

# 4. Content Description
4.1 Full Name Field
Allows users to enter their complete name, with a placeholder for guidance.
4.2 Email Field
Accepts a valid email address and is required for registration.
4.3 Mobile Number Field
Allows users to enter a mobile number with a placeholder indicating expected input.
4.4 Password Field
Allows users to create a password. The text is visible while typing for easier confirmation.
4.5 Confirm Password Fiels
Ensures users re-enter their password correctly.
4.6 User Type Dropdown
Allows the user to choose between "Student" or "Admin" for proper categorization.
4.7 Create Account Button
Submits the entered information to create a new account.
4.8 Login Link
Provides a message and a link for users who already have an account, redirecting them to the login page.

# 5. Summary
The Signup Page offers a clean, modern, and user-friendly interface for new users. It collects essential information using well-designed fields and interactive styling. Rounded corners, spacing, and shadows enhance the visual appearance, while the layout remains simple and easy to understand.

# Introduction
This README provides a detailed explanation of the Login Page structure and styling. The page is built using HTML and CSS, featuring a centered login box, input fields, a login button, and links for navigation such as "Forget Password" and "Signup".

# 2. Page Structure Overview
# 2.1 Head Section
The head section includes:
* The webpage title
* Meta tags for character support and responsive design
* Internal CSS for styling all elements of the login page
# 2.2 Body Section
The body contains:
* A centered outer box containing all login elements
* A heading for the login title
* Input fields for username and password
* A login button
* A link for forgotten passwords
* A message and link to navigate to the signup page

# 3. Styling Description (CSS)
# 3.1 Body Styling
* The default font is Arial for readability
* Background is light grey to highlight the login box
# 3.2 Outer Box Styling
* Fixed width for consistent layout
* Centered horizontally with top spacing using margin
* Padding for internal spacing around elements
* White background with rounded corners
* Shadow effect to create a floating card appearance
# 3.3 Heading Styling
* The heading is centered
* Uses purple color for visibility
* Extra spacing below the heading
# 3.4 Input Field Styling
* Fields span full width within the box
* Padding inside for comfortable typing
* Vertical spacing between fields
* Rounded corners with light grey border for modern design
# 3.5 Button Styling
* Full width button for easier interaction
* Purple background with white text
* Rounded corners with no border
* Background darkens slightly on hover
* Pointer cursor on hover
# 3.6 Link and Text Styling
* "Forget Password?" text is centered and blue
* Signup link is centered as a block element
* Message above signup link is black and centered
* Pointer cursor used for all clickable links
